  1. Hybrid Security Architecture Based on Virtually Abstracted Tools (VAT) for LLM Agent Data Loss Prevention
    Doyeong Gim, Junyeong Park, Junsik Kim, and 2 more authors
    In 2026 Annual Symposium of Kora Information Processing Society (ASK) , May 2026
    Abs DOI

    Autonomous agent systems powered by Large Language Models (LLMs) are maximizing interoperability with external databases through the adoption of the Model Context Protocol (MCP). However, this advancement has introduced new security challenges, including Data Over-Exposure (DOE), in which agents collect sensitive information beyond their functional requirements, and Tool-Orchestration Privacy Risks (TOP-R), where seemingly harmless fragments of information are combined to infer sensitive data. To address these threats, both academia and industry have primarily relied on post-processing approaches that employ LLMs as security controllers (LLM-as-a-Judge). Nevertheless, such approaches face inherent limitations, including hallucination and increased latency. To overcome these challenges and enable a next-generation Data Loss Prevention (DLP) architecture, this study proposes a four-stage hybrid security mechanism based on a Virtually Abstracted Tools (VAT) layer that enforces the Principle of Least Privilege (PoLP) from the design stage. The proposed conceptual architecture conceals original tools from the agent and exposes only fine-grained virtual atomic tools, thereby requiring the agent to explicitly demonstrate its intent through tool selection. Subsequently, the original data returned by the MCP server is forwarded to an LLM-as-a-Judge for post-processing, together with the selected virtual tool that provides contextual evidence of the agent’s intended operation. Through structural analysis, this paper demonstrates the logical validity of the proposed hybrid mechanism, showing its potential to significantly suppress hallucinations in LLM-as-a-Judge systems while reducing processing latency.

  2. Analysis of Data Over-Collection Threats in Agentic AI Environments
    Junsik Kim, Namhyun Ko, Doyeong Gim, and 2 more authors
    In 2026 Conference on Information Security and Cryptography (CISC) , May 2026
    Abs DOI

    Recently, Agentic AI has evolved beyond simple text generation to autonomously utilize external tools and resources. As a result, the adoption of the Model Context Protocol (MCP) has expanded rapidly, introducing new forms of data leakage threats that conventional security mechanisms are unable to detect effectively.This paper analyzes these emerging threats and identifies their root causes as excessive data provisioning by tools and the lack of contextual privacy awareness in Large Language Models (LLMs). In particular, the Broad Read approach, in which agent tools return data schemas that exceed operational requirements to maximize generality and reusability, creates opportunities for data over-collection, enabling LLMs to aggregate, infer, or disclose sensitive information. An examination of widely used MCP servers reveals that approximately 42.8% of existing designs inherently contain structural risks of over-collection. Furthermore, such risks are difficult to detect through traditional output-based monitoring mechanisms alone. Redesigning all existing APIs and databases according to an Atomic Read paradigm is practically infeasible due to the associated operational costs and system efficiency constraints. Therefore, rather than modifying the tools themselves, this paper proposes the introduction of a Context-Aware Filter layer between the MCP server and the agent. By analyzing the intent embedded in user prompts, the proposed approach enables context-sensitive control over data access and disclosure, thereby mitigating over-collection risks while preserving compatibility with existing systems.

  1. A Self-Synchronizing Cyber Deception Framework via Infrastructure as Code Reflection
    Junyeong Park, Sayeon Kim, Woohyun Jang, and 4 more authors
    In 2025 IEEE 30th Pacific Rim International Symposium on Dependable Computing (PRDC) , Nov 2025
    Abs DOI

    As cyberattacks become more sophisticated, the use of honeypots has emerged as an alternative to proactively collect attackers’ exploit strategies. However, conventional honeypots often lack realism and require much human effort to deploy and maintain in modern IT infrastructures. This excessive cost in resources creates a major obstacle to their widespread use. To address these challenges, this paper proposes a self-synchronizing cyber deception framework that upholds the declarative approach of Infrastructure as Code (IaC), maintaining idempotency and consistency while automatically generating a deceptive environment. Our framework treats the target system’s IaC files as a blueprint to automatically generate and deploy a high-fidelity, “digital twin” deception environment. Our framework uses an automated pipeline to analyze the IaC file, apply the predefined transformation rules, and dynamically build new container images - replicating structural elements while replacing the core application logic with a honeypot. After deployment, the framework keeps the deceptive environment synchronized with the original system by automatically re-deploying the pipeline in response to any changes in the source IaC file, increasing fidelity and reducing management costs. The implementation of this prototype successfully shows a reduction in the manual effort required for deploying and maintaining deception environments, presenting a scalable and sustainable framework for active defense. This provides a strong foundation for building the next-generation defense mechanisms that can adapt to both evolving cyberattacks and changing infrastructure.

  2. Automating Honeypots through Digital Twins and MITRE ATT&CK TTP Mapping
    Junyeong Park and Jonghyun Kim
    In 2025 Summer Conference on Information Security and Cryptography (CISC) , Jun 2025
    Abs DOI

    This study proposes and experimentally validates a Digital Twin Honeypot (DT Honeypot), an active cyber deception system designed to overcome the limitations of conventional honeypots by leveraging Digital Twin (DT) technology. To achieve this, the proposed system integrates the real-time synchronization capability of digital twins—which continuously replicate changes occurring in the original system to its virtual counterpart—with honeypot technology. To minimize the risk of sensitive data exposure resulting from this synchronization process, a data fictionalization module was designed and implemented. Furthermore, the system incorporates a real-time attack monitoring module that maps attacker activities to the Tactics, Techniques, and Procedures (TTPs) defined in the MITRE ATT&CK framework. Based on the identified attack patterns, the system dynamically exposes an appropriate layer of deceptive assets, thereby minimizing the scope of fake resources revealed to the attacker while maintaining the effectiveness of the deception strategy. Experimental results demonstrate that the DT Honeypot provides greater stability and flexibility than traditional static honeypots. Unlike conventional honeypots, the proposed system can adapt autonomously to changes in the original environment, significantly reducing maintenance costs associated with deception systems. Moreover, by addressing the challenges introduced by the use of digital twin technology, the DT Honeypot is shown to function effectively as an active and adaptive security mechanism.
