## Fizz Buzz

이 문제는 369 게임과 유사한 문제입니다. <br>
3의 배수일 때는 "Fizz"를, <br>
5의 배수일 때는 "Buzz"를, <br>
3과 5 모두의 배수일 때는 "FizzBuzz"를 배열에 저장하면 됩니다.<br>
그 외의 수에는 아무 대체도 하지 않고 그대로 정수를 문자형으로 저장하면 됩니다.<br>

<br>

- Input: n = 3
- Output: ["1","2","Fizz"]

- Input: n = 5
- Output: ["1","2","Fizz","4","Buzz"]

- Input: n = 15
- Output: ["1","2","Fizz","4","Buzz","Fizz","7","8","Fizz","Buzz","11","Fizz","13","14","FizzBuzz"]

<br>
<br>

이 문제의 핵심은 최종 반환 배열을 **문자열 배열인 \*\*array**로 선언하고, <br>
해당 문자열 배열에 삽입하기 위한 **sprintf** 함수와 **strdup** 함를 이용하는 것이었습니다.<br>
sprintf(<저장할 배열>, <문자형>, <삽입할 내용>) 구문을 이용해 임시 배열에 Fizz, Buzz, FizzBuzz를 저장하고, <br>
**strdup** 함수의 포인터 반환값을 이용해 반환할 문자열 배열에 저장합니다.<br>
<br>
이것만 제대로 잡혔다면 3과 5의 배수를 Fizz, Buzz, FizzBuzz로 치환하는 것은 별로 어렵지 않습니다.<br>
아래는 제가 작성한 코드이며,<br>
메모리 사용량에서 과하게 많은 메모리를 쓰고 있는 것으로 측정되어 해당 부분에서 최적화를 진행해볼 수 있을 것 같습니다.<br>
<br>
<hr/>

### C언어 작성 코드
```c
char** fizzBuzz(int n, int* returnSize) {
    *returnSize = n;
    char **array = malloc((*returnSize) * sizeof(char*));
    char temp[16];

    for (int i = 0; i < n; i++)
    {
        if ((i + 1) % 3 == 0 || (i + 1) % 5 == 0)
        {
            if ((i + 1) % 5 != 0)       // Only divisible by 3
            {
                sprintf(temp, "%s", "Fizz");
                array[i] = strdup(temp);
            }
            else if ((i + 1) % 3 != 0)
            {
                sprintf(temp, "%s", "Buzz");
                array[i] = strdup(temp);
            }
            else
            {
                sprintf(temp, "%s", "FizzBuzz");
                array[i] = strdup(temp);
            }
        }
        else
        {
            sprintf(temp, "%d", i + 1);
            array[i] = strdup(temp);
        }
    }
    return array;
}
```
